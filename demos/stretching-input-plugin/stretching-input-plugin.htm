Stretching Input Element jQuery Plugin

Easily make any input element stretch as the user types in to it, and squish as text is deleted.

Html & Inline Css:

   <input id="inputElement" type="text" style="width:30px">

JavaScript:

   $('#inputElement').stretchfield();

Plugin Features:

  • chainable jquery plugin
        $('#inputElement').stretchfield().fadeIn();
  • apply to single or multiple elements of a jquery selection
        $('input[type="text"').stretchfield();
  • re-apply the plugin to the same element(s) as much as you would like
        $('#inputElement').stretchfield({ maxWidth: '90px' });
        $('#inputElement').stretchfield({ maxWidth: '120px' });
  • continues to work its magic on existing user input after a page refresh or browser back/forward buttons
  • optionally set your input element's width, min-width, and/or max-width with css
        <input type="text" style="width:30px; min-width:30px; max-width:120px;">
  • optionally pass in 'minWidth' and/or 'maxWidth' parameters to the plugin constructor
        $('#inputElement').stretchfield({minWidth: '30px', maxWidth: '120px'});

Support, Testing, and Requirements:

  • supports IE7+, Chrome, Firefox, Safari and Opera
    • support is backed by extensive unit tests and manual UI testing
  • plugin requires jQuery and Modernizr JavaScript plugins
    • tested with: jQuery 1.7.1 (if using older versions of jQuery, please do so at your own risk)
    • tested with Modernizr 2.0.6 (if using older versions of Modernizr, please do so at your own risk)

Shorten Select Element jQuery Plugin

Option text that exceeds a select element's max-width will be truncated and '...' will be prepended to the beginning.

Check it out:

Note: Hovering your mouse over any shortened option text will reveal the complete option text in a tooltip (on browsers that support the 'title' attribute).

Html & Inline Css:

    <select style="max-width:350px;">
       <option value="0" disabled>Please select an item</option>
       <option value="1">Short item text</option>
       <option value="2">Long item text that is too long to fit inside a select option</option>
       <option value="3">Short item text</option>
    </select>

JavaScript:

   $('select').shortenSelect();

Plugin Features:

  • chainable jquery plugin
        $('#selectElement').shortenSelect().fadeIn();
  • apply to single or multiple elements of a jquery selection
        $('select').shortenSelect(); // applies to all select elements
  • re-apply the plugin to the same element(s) as much as you would like
        $('#selectElement').shortenSelect({ maxWidth: '350px' });
        $('#selectElement').shortenSelect({ maxWidth: '75px' });
        $('#selectElement').shortenSelect({ maxWidth: '350px' });
  • optionally set your input element's max-width with css
        <select style="max-width:350px;"> ...
  • optionally pass in 'maxWidth' parameter to the plugin constructor
        $('#selectElement').shortenSelect({ maxWidth: '350px' });

Support, Testing, and Requirements:

  • supports IE7+, Chrome, Firefox, Safari and Opera
    • support is backed by extensive unit tests and manual UI testing
  • plugin requires jQuery and Modernizr JavaScript plugins
    • tested with: jQuery 1.7.1 (if using older versions of jQuery, please do so at your own risk)
    • tested with Modernizr 2.0.6 (if using older versions of Modernizr, please do so at your own risk)
